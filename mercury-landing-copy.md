**How does pricing work?**

**Can Mercury transfer to a human?**  
Yes. Set rules for warm transfer based on intent, sentiment, keywords, or VIP callers. Mercury can pass a summary and context.

**Does it integrate with my CRM and tools?**  
Yes. Use native integrations where available or connect any REST API via tool calling.
